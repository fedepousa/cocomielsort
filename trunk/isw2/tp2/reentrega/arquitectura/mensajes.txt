

browser usuario externo a sistema web
Todos los mensajes desde browser usuario externo a sistema web tendrán los datos, incluído el checksum, encriptados con una clave simétrica, excepto el mensaje de log in. La clave simétrica será parte de la respuesta al mensaje de log in.  
header generico:
	tipo de mensaje
	(en todos menos en log in) numero de sesion
datos genéricos:
	checksum
tipos de mensaje
	log in
	cambio idioma
	postularse
	log out
	votar
	pedido de clave pública
datos segun tipo de mensajes
	log in
		nombre de usuario
		hash de contraseña
	cambio idioma
		nuevo idioma seleccionado
	postularse
		eleccion
	log out
		nombre de usuario
	votar
		eleccion
		voto (lista de candidatos concatenada a un 
			numero aleatorio, todo encriptado)
	pedido de clave pública
		eleccion
